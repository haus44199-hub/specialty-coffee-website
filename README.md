## ✨ Key Features

-   **Modern Visual Design:** Deep chocolate & caramel color scheme with gradient accents, backdrop-blur effects, and neon vibes.
-   **100% Responsive:** Flawlessly adapts to all screen sizes — from mobile phones to 4K displays (Mobile First approach).
-   **Dynamic Coffee Menu:** Coffee cards are rendered dynamically via JavaScript, making it easy to update or extend the menu.
-   **Smooth Animations:** Elements fade in as you scroll using Intersection Observer API.
-   **Interactive Contact Form:** Client-side validation with a custom toast notification system — no page reload.
-   **Custom Scrollbar:** Styled to match the brand's visual identity.
-   **No Frameworks:** Pure HTML5, CSS3, and vanilla JavaScript — no unnecessary dependencies.
